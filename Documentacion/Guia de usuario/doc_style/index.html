Escribir Documentación

Para ayudar a facilitar un estilo de documentación consistente y fácil de leer para los proyectos de CodeIgniter, Ellislab esta haciendo que se pueda disponer libremente de las etiquetas y los CSS de la guia del CodeIgniter para la comunidad y su uso. Para tu conveniencia, un archivo plantilla fue creado, este incluye las etiquetas del bloque primario usado junto con unos breves ejemplos.
