Solución de Problemas

Si descubres que pongas lo que pongas en la URL se carga siempre la página por defecto, podría ser que tu servidor no soporte la variable PATH_INFO necesaria para mostrar URLs amigables para los buscadores. Como primer paso, abre tu archivo application/config/config.php busca la información sobre URI Protocol. Te recomendará que pruebes con varias configuraciones alternativas. Si sigue sin funcionar tras haber intentado esto tendrás que forzar a CodeIgniter para que añada un signo de interrogación a las URLs. Para hacer esto abre tu archivo application/config/config.php y cambia esto:

$config['index_page'] = "index.php";

A esto:

$config['index_page'] = "index.php?";
