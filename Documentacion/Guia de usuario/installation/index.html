Instrucciones de Instalación

CodeIgniter se instala en cuatro pasos:

  1. Descomprimir el paquete.
  2. Subir todos los archivos y carpetas a su servidor. Habitualmente el archivo index.php estará en el root.
  3. Abrir el archivo application/config/config.php con un editor de texto y configurar la URL base. Si pretendes usar encriptación o sesiones, configura la clave de encriptación.
  4. Si pretende usar una base de datos, abra el archivo application/config/database.php con un editor de texto y configure los parámetros de la base de datos.

Si le gustaría aumentar la seguridad escondiendo los archivos de CodeIgniter puede renombrar las carpetas system y application a algo más privado. Si lo hace, debe abrir tu archivo index.php y configurar las variables $system_folder y $application_folder en la parte superior del archivo con los nuevos nombres que ha elegido.

Para obtener la mayor seguridad, ambas carpetas, tanto system como la carpeta application deberían colocarse por encima del root de manera que no sean directamente accesibles desde un navegador. Por defecto, se incluyen archivos .htaccess en cada carpeta para evitar el acceso directo, pero es mejor moverlas del acceso público por completo por si se diera el caso de que el servidor web cambia su configuración o no le afectan los archivos .htaccess.

Una vez movidos, abra su archivo index.php y configure las variables $system_folder y $application_folder, preferiblemente con la ruta completa, por ej. '/www/Mi_Usuario/system'.

Una medida adicional que se puede usar en entornos de producción es desactivar el reporte de errores PHP y culquier otra funcionalidad de desarrollo. En CodeIgniter, esto se puede hacer configurando la constante ENVIRONMENT, la cual está descrita en su totalidad en la página de seguridad.

¡Eso es todo!

Si es nuevo en CodeIgniter, por favor, lea la sección Primeros Pasos de la Guía del Usuario para comenzar a aprender como crear aplicaciones PHP dinámicas. Disfruta!
