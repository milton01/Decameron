Actualizar de Beta 1.0 to Beta 1.1

Para actualizar a Beta 1.1 por favor realice los siguientes pasos:

Paso 1: Reemplace su archivo índice

Remplace su archivo principal index.php con el nuevo archivo index.php. Nota: Si ha renombrado la carpeta "system" necesitará editar esta información en el nuevo archivo.

Paso 2: Reubique la carpeta de configuración

Esta versión de CodeIgniter ahora le permite a múltiples juegos de "applicaciones" compartir el juego de archivos comunes. Para permitir a cada aplicación tener sus valores de configuración propios, el directorio config debe residir dentro de la carpeta application, así que por favor muévalo aquí.

Paso 3: Remplace directorios

Remplace los siguientes directorios con las nuevas versiones:

  • drivers
  • helpers
  • init
  • libraries
  • scaffolding

Paso 4: Agregue el archivo de lenguaje de calendario

Hay un nuevo archivo de lenguaje correspondiente a la nueva clase de calendario la cual debe ser agregado a la carpeta de lenguaje. Agregue el siguiente item a tu versión: language/english/calendar_lang.php

Paso 5: Edite su archivo de configuración

El archivo original application/config/config.php tiene un error tipográfico en él. Abra el archivo y mire los items relacionados a las cookies:

$conf['cookie_prefix'] = "";
$conf['cookie_domain'] = "";
$conf['cookie_path'] = "/";

Cambie el nombre del arreglo de $conf a $config, así:

$config['cookie_prefix'] = "";
$config['cookie_domain'] = "";
$config['cookie_path'] = "/";

Por último, agregue los siguientes items al archivo de configuración (y edite la opcion si necesita):


/*
|------------------------------------------------
| URI PROTOCOL
|------------------------------------------------
|
| This item determines which server global
| should be used to retrieve the URI string. The
| default setting of "auto" works for most servers.
| If your links do not seem to work, try one of
| the other delicious flavors:
|
| 'auto' Default - auto detects
| 'path_info' Uses the PATH_INFO
| 'query_string' Uses the QUERY_STRING
*/

$config['uri_protocol'] = "auto";
